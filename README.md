# 支付宝商品批量上传工具

## 简介
这是一个帮助商家批量上传商品到支付宝的工具。主要功能包括：
1.  **素材上传**：批量上传图片到支付宝素材中心，获取图片链接。
2.  **商品配置**：上传Excel文件导入商品基础信息。
3.  **图片绑定**：可视化的方式将已上传的图片绑定到Excel中的商品。
4.  **批量创建**：一键将绑定好图片的商品发布到支付宝。

## 使用说明

### 1. 启动工具
双击运行 `run.bat`。
程序启动后，浏览器访问 `http://127.0.0.1:5000`。

### 2. 配置
在页面顶部输入您的支付宝商家后台 `Cookie` 和 `CToken`。
-   Cookie获取方式：登录商家中心，F12打开开发者工具，在网络请求中找到任意请求的Header中的Cookie。
-   CToken通常在Cookie中或者URL参数中能找到。

### 3. 素材图片上传
-   点击“选择文件”上传您需要使用的商品图片。
-   点击“上传选中图片”，工具会自动将图片上传到支付宝，并显示缩略图。

### 4. 商品配置 (Excel)
-   准备您的商品Excel文件。
-   Excel列名建议包含：`title` (标题), `price` (价格), `categoryId` (类目ID), `appId` (小程序ID) 等字段。
-   上传Excel后，页面会列出商品列表。

### 5. 绑定图片与发布
-   在商品列表中，点击“选择”按钮（主图、详情图、白底图）。
-   在弹出的窗口中勾选该商品对应的图片。
-   所有商品配置完成后，点击“批量创建商品”。
-   查看状态列确认是否发布成功。

## 注意事项
-   请确保Cookie有效。
-   Excel文件格式需要符合API要求的字段结构，工具主要负责图片链接的填充。
